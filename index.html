Prizer

Prizer is a skill-based mobile gaming application available in India. The platform allows users to participate in skill-based games where outcomes depend on user skill rather than chance.

User Authentication

Prizer uses SMS-based One-Time Password (OTP) verification to securely authenticate users during login and sensitive actions such as wallet access. OTP messages are sent only to users who explicitly request verification.

Key Features

  • Skill-based competitive games
  • Secure SMS OTP authentication
  • User wallet and transaction verification
  • India-only operations

Platform

Available on Android via Google Play Store.

Legal Disclaimer

Prizer is a skill-based game and does not involve gambling or betting. The application operates in compliance with applicable Indian laws.
